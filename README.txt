Ecommistry ProductList Module

A stand alone extension that displays chosen products in a new customer frontend page accessed from the My Account section.

1. Create a new Yes/No attribute for products "handle_display".
2. Displays a new page that only a logged in customer can access.
3. Add a new item link into the My Account menu connected to produclist module.
4. Displays product list with the handle_display attribute set to Yes.
